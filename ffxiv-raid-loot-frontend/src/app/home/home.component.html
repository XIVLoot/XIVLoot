Welcome to XIVLoot.com!

XIVLoot is a Gear Management Website that allows you and your static to easily distribute gear during an active raid tier. This project was created by a group of raiders who were tired of the old spreadsheets and janky ways groups managed loot drops, we hope to offer a more streamlined approach to accomplish this goal.

This website is still in active development but is functional on it's core features. Please join our discord server to see feature announcements, report any bugs and let us know what works and doesn't work as we shape this website to be a home for statics.

How to use XIVLoot.com

1. Login via discord or create an account via the Login button

2. Create a Static here
Please Note: Anyone with your statics unique URL can view your statics page.
Once you Click the create button, please fill out the name and start populating members. We recommend claiming your character so it is locked to your account.

3. Edit Players Names/ Jobs and Gear
We have provided tie-ins to Etro and XIVgear links to allow you to easily import your BIS lists.

4. You can modify your statics settings such as.. Player Gear Score (PGS), Lock Settings, etc..

If you have any questions regarding the use of this website, please stop by our discord where we will be happy to help.
